
Математический кружок.

В математическом кружке ученики решают задачи разной сложности, помогают друг другу. Для базы нужно следить, какие задачи решил каждый ученик, 
помечать, кто кому объяснял задачи. База данных будет содержать следующие сущности: 

узлы:
1. Ученик (уникальный номер ученика, имя, класс, уникальный номер его учителя);
2. Задача (уникальный номер задачи, название задачи, тема (алгебра, геометрия), сложность (1,...,10);
3. Тема (уникальный номер темы, название темы);
4. Решение задачи (уникальный номер события, дата решения, время, потраченное на задачу);

рёбра:
1. Решает: ученик->задача (решил правильно или решил неправильно);
2. Объясняет: ученик->ученик (дата объяснения);
3. Относится к теме: задача->тема (тема является основной или необязательной);
4. Требует знаний: тема->тема (минимальный уровень знаний для понимания)
